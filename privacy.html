2. Erhebung und Speicherung personenbezogener Daten

2.1 Spielername

Beim ersten Start der App werden Sie aufgefordert, einen Spielernamen einzugeben. Dieser Name wird ausschließlich lokal auf Ihrem Gerät gespeichert (localStorage) und dient zur Personalisierung Ihrer Spielerfahrung.

2.2 Highscores

Wenn Sie ein Spiel beenden, wird Ihr Score zusammen mit Ihrem Spielernamen an unseren Server übermittelt und in einer Datenbank gespeichert, um die Bestenliste zu erstellen. Diese Daten sind öffentlich einsehbar und enthalten:

  • Spielername (von Ihnen gewählt, max. 12 Zeichen)
  • Erreichte Punktzahl

Es werden keine weiteren personenbezogenen Daten (E-Mail, IP-Adresse, etc.) mit den Scores gespeichert.

2.3 Service Worker & Caching

Die App nutzt einen Service Worker, um Inhalte lokal auf Ihrem Gerät zu speichern. Dies ermöglicht die Offline-Funktionalität. Gespeichert werden ausschließlich:

  • App-Code (HTML, CSS, JavaScript)
  • Grafische Assets
  • Temporär: Ausstehende Score-Submissions (bei Offline-Nutzung)

Diese Daten können Sie jederzeit über die Browser-Einstellungen löschen.

3. Verwendung von Analysetools

Wir nutzen Vercel Analytics zur anonymisierten Erfassung von Nutzungsstatistiken (Seitenaufrufe, Gerätekategorien). Es werden keine personenbezogenen Daten erfasst oder mit Dritten geteilt. Vercel Analytics verwendet keine Cookies.

4. Hosting & Server

Die App wird gehostet über:

  • Vercel Inc. (Frontend Hosting) - Server in der EU
  • Redis Cloud (Highscore-Datenbank) - Server in der EU (Frankfurt)

5. Ihre Rechte

Sie haben das Recht auf:

  • Auskunft: Welche Daten über Sie gespeichert sind
  • Berichtigung: Korrektur falscher Daten
  • Löschung: Entfernung Ihrer Daten aus der Highscore-Liste
  • Widerspruch: Widerspruch gegen die Datenverarbeitung

Um Ihre Rechte auszuüben, kontaktieren Sie uns unter der oben genannten E-Mail-Adresse. Geben Sie dabei Ihren Spielernamen an, damit wir Ihre Daten identifizieren können.

6. Datenlöschung

Sie können Ihre lokal gespeicherten Daten jederzeit löschen durch:

  • App deinstallieren
  • Browser-Cache leeren
  • In der App: "Abmelden / Anderer Fahrer" klicken

Ihre Highscores in der Bestenliste können Sie nicht selbst löschen. Kontaktieren Sie uns hierfür per E-Mail.

7. Keine Weitergabe an Dritte

Ihre Daten werden nicht an Dritte verkauft oder weitergegeben, außer:

  • Gesetzliche Verpflichtung besteht
  • Zur technischen Bereitstellung notwendig (Hosting-Provider)

8. Sicherheit

Wir setzen technische und organisatorische Sicherheitsmaßnahmen ein, um Ihre Daten gegen Manipulation, Verlust oder unbefugten Zugriff zu schützen:

  • HTTPS-Verschlüsselung für alle Datenübertragungen
  • Eingabevalidierung (max. 12 Zeichen, keine Sonderzeichen)
  • Rate Limiting für API-Anfragen
